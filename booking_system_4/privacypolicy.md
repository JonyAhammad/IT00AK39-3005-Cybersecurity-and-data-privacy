## 1. About This Policy
This policy explains how the Booking System uses and protects personal data according to GDPR.

## 2. Personal Data We Collect
We collect only necessary data:
- Email address (username)
- User role (reserver or administrator)
- Birthdate (to check age over 15)
- Reservation details (resource and time)
- Login IP address and time
- Session and security tokens

## 3. Why We Use This Data
Personal data is used to:
- Log users into the system
- Manage bookings and resources
- Check age requirement
- Protect the system from misuse

## 4. User Consent
Users give consent by accepting the terms during registration.

## 5. Data Protection
- Passwords are safely stored using hashing
- CSRF protection is used
- Access is controlled by user roles

## 6. Cookies
The system uses essential cookies for login sessions and security. These cookies are required for the system to work.

## 7. User Rights
Users can:
- View and update their personal data
- Ask the administrator to delete their account
- Withdraw consent by requesting account removal

## 8. Data Sharing
Personal data is not shared with third parties.

## 9. Changes
This policy may be updated if needed.

## 10. Contact
For privacy questions, users can contact the system administrator.
